# João Pedro Ribeiro

**Brazilian Army Sergeant → Networking & Security | DevSecOps (in progress)**

I spent 8 years in the Brazilian Army, where I support my unit's IT section with the battalion's network and Linux servers — switches, cabling, servers, and support tickets. I'm now building a strong foundation in **networking and security**, the base of a DevSecOps career, and growing into automation on top of it.

I study daily and document my learning journey openly on GitHub.
